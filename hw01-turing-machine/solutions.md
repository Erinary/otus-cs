### Машина Тьюринга

#### 1) Проверка, какое число символов передано - четное или нечетное

Если четное, добавляет в конце "E", если нечетное - "O".

```
; State 0: read the leftmost symbol
0 * * r 1
0 _ E * halt

1 * * r 0
1 _ O * halt
```

#### 2) Инкремент числа на 1.

Реализация для двоичной системы.

```
; State 0: read the leftmost symbol
0 * * r 0
0 _ _ l q1

q1 0 1 * halt
q1 1 0 l q1
q1 _ 1 * halt
```
