```
Y = (ABCD + ABCD + ABCD) + ABC + ABD + BCD + A = 
= (ABCD + ABC) + ABD + BCD + A =
= (ABC + A) + ABD + BCD = 
= (A + ABD) + BCD = 
= A + BCD
```
